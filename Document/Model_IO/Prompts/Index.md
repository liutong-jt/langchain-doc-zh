# Prompts

语言模型的提示是用户为引导模型响应而提供的指令或输入，帮助它理解上下文并生成相关且连贯的基于语言的输出，例如回答问题、完成句子或进行对话。

## 快速入门

此[快速入门](https://python.langchain.com/docs/modules/model_io/prompts/quick_start)提供了如何处理提示的基本概述。
